Welcome to the MS Scratch and HS Programming Final Projects

Crossroads Homeschool Co-Op {{page.school-year}}

The Middle School class used the program called Scratch, which is a drag and drop programming language developed by MIT.

This year, the students used an online video resource created by Kristian Mischke to fascilitate individualized learning. Throughout the year, the students learned how to make creative interactive projects, including:

  • An interactive project that shared about their interests
  • A basic game with a main menu
  • Other small challenges
  • And their final projects are displayed further down on this page


The Highschool class learned a programming environment called Processing which built on the Java programming language
